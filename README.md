# DocAppointment

DocAppointment is a modern and responsive doctor appointment booking platform that helps patients easily find doctors, explore detailed profiles, and book appointments online in just a few clicks.

## Live Site
https://docappoint-client.vercel.app

---

## ✨ Key Features

- 🔐 Secure Authentication System with user login & logout functionality
- 👨‍⚕️ Doctor Details Page with specialization, experience, hospital, rating, and appointment fee
- 📅 Easy Appointment Booking system with date & time selection modal
- 📝 Dashboard Management to view, update, and delete booked appointments
- 👤 User Profile Section with profile information management
- ⚡ Fully Responsive UI optimized for mobile, tablet, and desktop devices
- 🎨 Modern & Clean Design built with Tailwind CSS
- 🔄 Real-time Appointment Update functionality from dashboard
- 🚀 Fast and smooth deployment using Vercel
